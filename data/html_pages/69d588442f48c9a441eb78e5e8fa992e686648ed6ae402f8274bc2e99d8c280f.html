OFFICIAL APPLICATION
Welcome to UCR's Graduate Application System (GradSIS). The system is no longer accepting applications for graduate study. However, applicants who applied for Summer/Fall 2022, Winter 2023, and Spring 2023 before September 13, 2022, will have access to review their application status and provide a decision to an offer of admission through this portal.

All saved applications in this system are no longer available to access or submit. Please visit our new Graduate Application Portal to create an account and start a new application.

Mozilla Firefox and Internet Explorer are the best web browsers to use for our application system. Other browsers like Google Chrome, have not been tested to confirm that GradSIS can support them. For more details, visit our Technical Considerations page. If you require assistance within the GradSIS application system, please select the Technical Problems/Feedback link at the bottom of each page.
